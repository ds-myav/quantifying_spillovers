## Abstract
<p align="justify">
Promotional activities play a key role in retail marketing, yet their influence often extends beyond the promoted items. Quantifying these indirect effects within a product category is essential for effective pricing, shelf allocation, and inventory management, but remains challenging due to the complex interdependence of consumer responses.
This study proposes a two-stage methodology for the causal estimation of within-category spillover effects generated by price promotions in retail environments. 
The first stage develops an advanced data processing framework to uncover many-to-one relationships between promoted and non-promoted products and to construct a feature-rich representation of observational retail data. This enables the second stage to causally estimate spillover effects through a DML approach, complemented by empirical diagnostic checks. The methodology is examined under three alternative DML specifications and evaluated using 42 months of daily data from the Carbonated Soft Drinks category of a large Italian grocery retailer. Results indicate that fully linear models fail to capture the complexity of promotional interactions, whereas flexible nonparametric approaches yield statistically significant effects generating average sales variations ranging from -36.96% to +25.65%.
The findings reveal predominantly negative spillovers, reflecting cannibalization and inter-brand competition in line with established marketing research, alongside limited evidence of complementarity under context-dependent conditions, as highlighted through model explainability analysis.
Placebo permutation tests provide additional support against spurious findings in the observational setting. Overall, the study combines causal inference with explainable analytics, offering actionable insights for promotion management and category planning.
</p>

## Usage

This repository provides an automated script to reproduce the Python environment and execute the experiment locally.<sup>(1)</sup>

### Local execution

Follow these steps to reproduce the experiment on your machine:

1. **Clone the repository**
   
   ```
   git clone https://github.com/ds-myav/quantifying_spillovers.git
   ```

2. **Setup and activate the environment**
   
   Navigate to the `quantifying_spillovers/` folder and execute the provided shell script to create and activate the virtual environment, install dependencies, and register the Jupyter kernel:
   ```
   cd quantifying_spillovers
   chmod +x activate_env.sh
   ./activate_env.sh
   ```

3. **Run the experiment**
   
   Execute all Jupyter notebook cells to reproduce the results presented in the study.<sup>(2)</sup>

---
<sup>(1)</sup> *This repository requires Python 3.9. Any compatibility issues with later versions must be resolved locally by the user.* <br />
<sup>(2)</sup> *Due to inherent nondeterminism in computation, the numerical results obtained when reproducing the experiment may show slight variations compared to those reported in the paper, depending on hardware differences.*
